# 🎮 Free Games Explorer

A Vue 3 application to explore free-to-play games using the FreeToGame API.

---

## 🚀 Features

* 🔍 Search games
* 🎮 Filter by genre
* 💻 Filter by platform
* 🔤 Sort games
* ❤️ Favorites (saved in localStorage)
* ⚡ Infinite scroll
* 🎨 Responsive UI with Tailwind
